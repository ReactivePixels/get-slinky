Slinky takes the hard work out of managing your npm link dependencies.

With Slinky you can link and unlink your npm dependencies with a single click!

Get started.

Install Slinky

Slinky is available in 3 flavours for Windows, OSX and Linux. Download and install the appropriate version for your platform.

2

Link your Packages

You will need to symlink the modules you would like to use. Check out the npm documentation for further details.

cd path/to/my-npm-package
npm link
3

Start Using Slinky

Now you are ready to start using Slinky.

  1. Open Slinky from your task tray.
  2. Select the folder where your Project lives.
  3. Slinky will show you what packages you have available to start 'linking' and 'unlinking'.

Slinky is a completely open source project powered by Electron. Feel free to view, clone and contribute.
